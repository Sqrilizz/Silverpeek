## 📄 Описание файлов

### Основные файлы

#### `recipes.yml`
**Самый важный файл!** Содержит все рецепты в формате YAML для плагина BreweryX.

**Использование:**
```bash
# Скопируйте в папку плагина
cp recipes.yml plugins/BreweryX/recipes.yml

# Перезагрузите конфигурацию
/brew reload
```

#### `README.md`
Главная страница проекта с:
- Описанием проекта
- Быстрым стартом
- Навигацией по категориям
- Статистикой
- Полезными ссылками

#### `banner.png`
Главный баннер проекта, отображается в README.md

## 🔄 Рабочий процесс

### Для игроков:
1. Читайте `00_гайд_по_пивоварению.md`
2. Выбирайте рецепты из категорий (`01_пиво.md` - `05_безалкогольные.md`)
3. Варите напитки на сервере!

### Для администраторов:
1. Скачайте `recipes.yml`
2. Скопируйте в `plugins/BreweryX/recipes.yml`
3. Выполните `/brew reload`
4. Готово!

### Для контрибьюторов:
1. Форкните репозиторий
2. Добавьте рецепт в `recipes.yml`
3. Обновите соответствующий `.md` файл
4. Протестируйте на сервере
5. Создайте Pull Request
