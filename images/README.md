# 🖼️ Изображения для книги рецептов

Эта папка содержит все изображения, используемые в книге рецептов.

## 📋 Список необходимых изображений

### Основные изображения:
- ✅ `banner.png` - Главный баннер (уже добавлен в корень проекта)
- ⬜ `brewing_banner.png` - Баннер для гайда по пивоварению
- ⬜ `features.png` - Иллюстрация возможностей BreweryX
- ⬜ `cheers.png` - Изображение для завершения страниц

### Оборудование:
- ⬜ `cauldron.png` - Котёл над огнём для варки
- ⬜ `brewing_stand.png` - Зельеварка для дистилляции
- ⬜ `barrels.png` - Различные типы бочек

### Процессы:
- ⬜ `cooking.png` - Процесс варки в котле
- ⬜ `distillation.png` - Процесс дистилляции
- ⬜ `aging.png` - Процесс выдержки в бочке

### Качество и эффекты:
- ⬜ `quality.png` - Уровни качества напитков
- ⬜ `drunk.png` - Эффекты опьянения

## 📐 Рекомендуемые размеры

- **Баннеры:** 1200x300 px
- **Иллюстрации процессов:** 800x600 px
- **Иконки оборудования:** 400x400 px
- **Декоративные элементы:** 200x200 px

## 🎨 Стиль

- Используйте скриншоты из Minecraft
- Добавляйте текстовые пояснения при необходимости
- Сохраняйте в формате PNG для прозрачности
- Оптимизируйте размер файлов

## 📝 Как добавить изображения

1. Создайте или сделайте скриншот нужного изображения
2. Оптимизируйте размер (можно использовать TinyPNG)
3. Сохраните в эту папку с соответствующим именем
4. Убедитесь, что ссылки в markdown файлах корректны

## 🔗 Использование в markdown

```markdown
![Описание](images/имя_файла.png)
```

---

*Все изображения должны быть связаны с Minecraft и BreweryX плагином*
