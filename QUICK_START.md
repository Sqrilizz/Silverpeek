# ⚡ Быстрый старт

Краткое руководство для быстрого начала работы с книгой рецептов BreweryX.

## 🎮 Для игроков

### Шаг 1: Изучите основы
Прочитайте [Гайд по пивоварению](00_гайд_по_пивоварению.md) - там всё подробно объяснено!

### Шаг 2: Выберите рецепт
Начните с простых рецептов:
- [🍺 Пшеничное пиво](01_пиво.md#пшеничное-пиво) - сложность ⭐
- [🍞 Квас](05_безалкогольные.md#квас) - сложность ⭐⭐
- [🍲 Картофельный суп](05_безалкогольные.md#картофельный-суп) - сложность ⭐

### Шаг 3: Подготовьте оборудование
Вам понадобится:
- 🔥 Котёл над огнём
- 🕐 Часы (для проверки времени)
- 🍶 Стеклянные бутылки
- 📦 Ингредиенты из рецепта

### Шаг 4: Варите!
1. Наполните котёл водой
2. Добавьте ингредиенты (ПКМ по котлу)
3. Ждите нужное время (проверяйте часами)
4. Соберите бутылками

## 🛠️ Для администраторов

### Установка за 3 шага:

```bash
# 1. Скачайте BreweryX
wget https://hangar.papermc.io/BreweryTeam/BreweryX/versions/latest/download -O BreweryX.jar

# 2. Установите плагин
mv BreweryX.jar plugins/

# 3. Скопируйте рецепты
cp recipes.yml plugins/BreweryX/recipes.yml

# Перезапустите сервер или выполните:
/brew reload
```

### Готово! 🎉

Теперь игроки могут варить напитки по рецептам из этой книги.

## 🆘 Частые вопросы

**Q: Как проверить время варки?**  
A: ПКМ по котлу с часами в руке

**Q: Почему напиток получился плохого качества?**  
A: Проверьте точность времени варки, количество дистилляций и выдержку

**Q: Где взять синие цветы для джина?**  
A: Василёк, орхидея или другие синие цветы

**Q: Как отрезвиться?**  
A: Выпейте кофе (-6%) или холодный кофе (-8%)

**Q: Что делать с опьянением 100%?**  
A: Вы "проснётесь" в точке пробуждения после выхода с сервера
