# 🤝 Руководство по внесению вклада

Спасибо за интерес к проекту! Мы рады любому вкладу в развитие книги рецептов BreweryX.

## 📋 Как внести вклад

### 1. Добавление новых рецептов

Если у вас есть интересный рецепт:

1. **Форкните репозиторий**
2. **Создайте ветку** для вашего рецепта:
   ```bash
   git checkout -b recipe/название-напитка
   ```

3. **Добавьте рецепт в `recipes.yml`:**
   ```yaml
   recipe_id:
     name: Плохое/Среднее/Отличное название
     ingredients:
       - INGREDIENT/количество
     cookingtime: минуты
     distillruns: количество
     distilltime: секунды
     wood: тип_дерева
     age: годы
     color: цвет
     difficulty: 1-10
     alcohol: процент
     lore:
       - Описание
     effects:
       - EFFECT/уровень/длительность
   ```

4. **Добавьте описание в соответствующий `.md` файл:**
   - Пиво → `01_пиво.md`
   - Вино → `02_вино.md`
   - Крепкие напитки → `03_крепкие_напитки.md`
   - Ликёры → `04_ликёры.md`
   - Безалкогольные → `05_безалкогольные.md`

5. **Протестируйте рецепт** на сервере с BreweryX

6. **Создайте Pull Request**

### 2. Улучшение документации

- Исправление опечаток
- Улучшение описаний
- Добавление советов и хитростей
- Перевод на другие языки

### 3. Добавление изображений

- Скриншоты процессов
- Иллюстрации оборудования
- Декоративные элементы

См. `images/README.md` для требований к изображениям.

## ✅ Требования к рецептам

### Обязательно:
- ✅ Рецепт должен работать в BreweryX
- ✅ Уникальный ID рецепта
- ✅ Понятное название на русском
- ✅ Корректные ингредиенты
- ✅ Указана сложность (1-10)

### Желательно:
- 📝 Подробное описание для разных уровней качества
- 🎨 Интересные эффекты зелий
- 💡 Советы по приготовлению
- 🎯 Тематическое соответствие

## 📝 Стиль написания

### Для YAML (recipes.yml):
- Используйте snake_case для ID
- Комментарии на русском языке
- Отступы: 2 пробела

### Для Markdown (.md файлы):
- Заголовки с эмодзи
- Структурированные секции
- Понятные описания
- Советы в блоках с 💡

## 🧪 Тестирование

Перед отправкой PR:

1. **Проверьте YAML синтаксис:**
   ```bash
   yamllint recipes.yml
   ```

2. **Протестируйте на сервере:**
   - Установите BreweryX
   - Загрузите ваш рецепт
   - Сварите напиток
   - Проверьте все этапы
   - Убедитесь в корректности эффектов

3. **Проверьте markdown:**
   - Все ссылки работают
   - Изображения отображаются
   - Форматирование корректно

## 🐛 Сообщение об ошибках

Нашли ошибку? Создайте Issue с:

- **Описанием проблемы**
- **Шагами для воспроизведения**
- **Ожидаемым поведением**
- **Скриншотами** (если применимо)
- **Версией BreweryX**

## 💬 Вопросы и предложения

- Опишите вашу идею подробно
- Приложите примеры, если возможно
