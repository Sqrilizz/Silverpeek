## 🍳 Как варить

Каждый напиток проходит до трёх этапов. **Не переварите и не передистиллируйте** — это портит качество.

### Этап 1 — Варка
1. Котёл над огнём / лавой / магма-блоком / костром → наполните водой
2. ПКМ с ингредиентами по котлу — добавьте все из рецепта
3. **ПКМ с часами** по котлу — проверить сколько прошло времени
4. ПКМ с пустыми бутылками — собрать напиток (до 3 шт. за раз)

### Этап 2 — Дистилляция *(только для некоторых)*
1. Бутылки → в зельеварку + **огненный порошок** как топливо
2. Каждый цикл = 1 дистилляция, следите за счётчиком из рецепта

### Этап 3 — Выдержка *(только для некоторых)*
1. Постройте бочку из нужного дерева → ПКМ по табличке
2. Положите бутылки, закройте инвентарь
3. Ждите: **1 год = 20 минут** реального времени (стареет даже оффлайн)

## 🛠️ Оборудование

### Котёл
- Любой котёл над огнём / лавой / магма-блоком / костром
- Часы (ПКМ по котлу) — проверить время варки

### Зельеварка
- Стандартная зельеварка + огненный порошок

### Бочки

**Малая (9 слотов):**
```
Слой 1: Ступеньки по углам
Слой 2: Доски по углам
```

**Большая (27 слотов):**
```
Слой 1: 5× Ступеньки (крест)
Слой 2: 5× Доски (крест)
```

## ⭐ Качество и опьянение

### Качество

| Качество | Как получить |
|----------|-------------|
| ⭐ Плохое | Неточное время / неверное количество / плохая выдержка |
| ⭐⭐⭐ Среднее | Рецепт выполнен правильно |
| ⭐⭐⭐⭐⭐ Отличное | Всё идеально (±10% по времени варки) |

### Опьянение

| Уровень | % | Эффекты |
|---------|---|---------|
| � Лёгкое | 1–30% | Покачивание камеры, редкие опечатки |
| � Среднее | 31–60% | Шатание, частые опечатки, тошнота |
| 🟠 Сильное | 61–90% | Сильное шатание, искажение чата, слабость |
| 🔴 Критическое | 91–100% | Потеря сознания, пробуждение в точке спауна |

**Отрезвление:** ☕ Кофе (−6%) · 🧊 Холодный кофе (−8%) · ⏰ Время

### Команды
```
/brew info              — ваш уровень опьянения
/brew info <игрок>      — опьянение игрока
/brew help              — помощь
```

## 🤝 Добавить рецепт

1. Форкните репозиторий → создайте ветку `recipe/название`
2. Добавьте рецепт в `recipes.yml`:

```yaml
recipe_id:
  name: Плохое/Среднее/Отличное название
  ingredients:
    - INGREDIENT/количество
  cookingtime: минуты
  distillruns: количество
  distilltime: секунды
  wood: тип_дерева
  age: годы
  alcohol: процент
  difficulty: 1-10
  effects:
    - EFFECT/уровень/длительность
```

3. Обновите соответствующий `.md` файл рецептов
4. Проверьте YAML: `yamllint recipes.yml`
5. Создайте Pull Request
